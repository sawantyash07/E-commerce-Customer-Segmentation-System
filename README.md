SmartCart Clustering System

The SmartCart Clustering System is a machine learning project that focuses on intelligent customer segmentation for an e-commerce platform. It uses unsupervised learning techniques to group customers based on their demographics, purchasing behavior, and engagement patterns.

Objective
The main goal of this project is to replace generic marketing strategies with data-driven, personalized marketing by identifying different customer groups such as high-value customers and churn-prone users.

Dataset Overview

Contains 2240 customer records with 22 attributes
Includes customer demographics, purchase amounts, purchase frequency, and engagement data

Key Features

Customer demographic analysis (age, income, education, marital status)
Purchase behavior analysis (product spending and buying frequency)
Customer engagement tracking (website visits, recency, complaints)
Customer segmentation using clustering algorithms

Technology Used

Python
Machine Learning (Unsupervised Learning)
Clustering Algorithms (e.g., K-Means)

Outcome

The system helps businesses:
Understand hidden customer behavior patterns
Improve customer retention
Design targeted and effective marketing campaigns
